
## Installation

### Create an environment
Whatever you prefer (e.g. `conda` or `venv`)
```console
pip install virtualenv
virtualenv venv
```

### Activate it
Mac / Linux:
```console
. venv/bin/activate
```
Windows:
```console
venv\Scripts\activate
```
### Install  dependencies
 pip install -r requirements.txt
 
## Usage
Run
```console
python app.py
```
